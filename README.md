# CS2 Hitsounds Pack

A curated **CS2 hitsound collection** focused on clarity, consistency, and fast audio feedback.

## Features

* **179 unique hitsounds**
* Clean, punchy, digital, and HVH-style sounds
* Consistent loudness (normalized)
* Low latency, no clipping

## Compatibility

* Counter-Strike 2 (CS2)
* **File type:** VSND_C (CS2 compiled sound)
* Engine-ready, no conversion needed

## Usage

Place desired hitsounds into your CS2 hitsound directory and select them via in-game settings or config.
